### The correlation files are stored in the following format:

### "20q_la{c}_{int_type}_{kappa_value}_iter{iteration}.pkl"

### where c is the number of cycles = {5,9,20},
### int_type = {ZZ, XX, XY, XXX, XZ} corresponding to {ising, ising-X, XX+YY, XX+YY+ZZ, XX+ZZ}, 
### kappa_value = {1,2,4},
### iteration = {1,2,3,4,5}.
